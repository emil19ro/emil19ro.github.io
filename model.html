MERN Product Management App

This app allows admins to manage products in a seamless way, providing an easy-to-use interface to add, update, and delete products. It includes two themes for better customization: Dark and Light modes. Chakra UI is used for a smooth and responsive user experience.

  • Dark and Light Themes: Switch between dark and light themes with ease to fit your preference.
  • CRUD Product Management: Admins can efficiently add, update, or delete products stored in a MongoDB database.
  • Responsive UI: Chakra UI ensures a smooth user interface, optimized for both desktop and mobile devices.
